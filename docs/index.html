About This Project

This site documents the ongoing decompilation of Toy Story 2: Buzz Lightyear to the Rescue. The goal is a clean, well-commented C++ source reconstruction that compiles to a matching binary (function parity at first) and serves as a readable reference for the game's engine and asset pipeline.
